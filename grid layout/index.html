Propiedades de Alieneación

Propiedades de Alienación de los items

Justify-items: it's a property that is used to align items inside the grid in a horizontal way

Align-items: it's a property that is used to align items inside the grid in a vertical way

Place-items: this is the combination of the last properties

Propiedades de Alienación de la grilla

Justify-content: it's a property that is used to align the grid in a horizontal way

Align-content: it's a property that is used to align the grid in a vertical way

Place-content: this is the combination of the last properties

Propiedades de Alienación de un objeto hijo dentro de la grilla

Justify-self: it's a property that is used to align the item it self in a horizontal way

Align-self: it's a property that is used to align the item it self in a vertical way

Place-self: this is the combination of the last properties
